### GeoAnalyticApp (Frontend)

# 📌 Описание

GeoAnalyticApp — демонстрационное гео-аналитическое приложение, предназначенное для отображения точек предприятий на карте, а также анализа данных и формирования рекомендаций.

# Frontend реализован на React и взаимодействует с backend-сервисом для получения данных, аутентификации пользователей и выполнения аналитических расчётов.

⸻

# 🚀 Основной функционал
 • 📍 Отображение точек на карте
 • 📊 Бизнес-аналитика:
 • плотность
 • конкуренция
 • дистанция
 • 🧠 Формирование рекомендаций
 • 🔐 Аутентификация и авторизация
 • 🔑 Поддержка двухфакторной аутентификации (2FA)
 • 👤 Управление профилем пользователя

⸻

# 🛠️ Используемые технологии
 • React
 • TypeScript
 • Redux Toolkit
 • React Query
 • Axios
 • Formik + Yup
 • React Bootstrap
 • JWT (jwt-decode)

⸻

# 🗺️ Карты

# В проекте используется библиотека:
 • Yandex Maps API v2.1 через @pbe/react-yandex-maps

# ⚠️ Важно:
 • Используется legacy API (v2.1)
 • В проекте необходимо учитывать устаревшие подходы (legacy-решения)
 • Используется бесплатный API-ключ, предназначенный исключительно для демонстрации
